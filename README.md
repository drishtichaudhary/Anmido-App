# Anmido 🌌

**Anmido** is a lightweight Flutter animation playground app built to explore and understand core animation techniques in Flutter. The app features a bouncing circular button with radius transitions, a continuously rotating star icon, and a smooth background scale effect.

---

## ✨ Features

- 🎯 **Animated Button**: Radius transitions using `AnimatedContainer` and gesture detection  
- 🌟 **Rotating Icon**: Infinite rotation of a star icon via `AnimatedBuilder` and `AnimationController`  
- 🎨 **Background Animation**: Smooth background scaling using `TweenAnimationBuilder`

---

## 🛠️ Tech Stack

- **Framework**: Flutter  
- **Language**: Dart  
- **Widgets Used**:  
  - `AnimatedContainer`  
  - `TweenAnimationBuilder`  
  - `AnimatedBuilder`  
  - `AnimationController`  
  - `GestureDetector`

## 🚀 Getting Started

1. **Clone the repo**

```bash
git clone https://github.com/your-username/anmido.git
cd anmido
```

2. **Install dependencies**
```bash
flutter pub get
```

3. **Run the app**
```bash
flutter run
```

## 💡 Purpose
- This app is created as a self-learning project to:
- Practice Flutter’s built-in animation APIs
- Understand stateful interactions and performance-safe animation handling
- Experiment with motion-based UI components
